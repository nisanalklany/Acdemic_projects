# Pneumonia Detection (Deep Learning)

## Overview
Developed a deep learning-based diagnostic system for detecting pneumonia from chest X-ray images.

## Approach
- Implemented and compared custom CNN architectures with Transfer Learning models (ResNet, DenseNet)
- Used TensorFlow for model development and training
- Applied data augmentation techniques to improve generalization
- Performed hyperparameter tuning (learning rate, batch size, etc.)

## Evaluation
- Focused on F1-score to balance sensitivity and precision for medical relevance
- Compared model performance across different architectures

## Results
- Achieved high classification performance on validation data
- Demonstrated the effectiveness of transfer learning over custom CNN in this task

## Key Learnings
- Practical experience with deep learning pipelines (data → training → evaluation)
- Understanding of trade-offs between precision and recall in medical applications
- Hands-on work with model optimization and performance tuning
